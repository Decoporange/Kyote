# 共通テスト 成績管理

共通テスト過去問の成績を記録・分析するWebアプリです。  
外部ライブラリ不要の純粋なHTML/CSS/JSで動作します。

## 機能

- **9教科対応**：英語R/L・国語（200点満点）・数学ⅠA/ⅡBC・物理・化学・地理・情報
- **年度管理**：自由ラベルで複数年度を追加・削除
- **大問管理**：各年度に大問を追加（デフォルト「第N問」）、得点/満点から得点率を算出
- **教科ごとの満点に基づく得点率**：国語は200点満点で自動換算
- **年度選択式 総合得点**：概要ページで年度を選択して合計を確認
- **グラフ**：教科別得点推移・教科別得点率 全年度比較（チェックボックス切替）
- **ダークモード**・教科カラーカスタマイズ
- **JSON エクスポート/インポート**でデータ管理
- `localStorage` による自動保存

## 使い方

### GitHub Pages でホスティング（推奨）

1. このリポジトリを GitHub に push する
2. **Settings → Pages → Source** で `main` ブランチの `/ (root)` を選択
3. `https://<あなたのユーザー名>.github.io/<リポジトリ名>/` でアクセス可能

### ローカルで開く

```
index.html をブラウザで直接開くだけで動作します。
```

## データについて

データは `localStorage` にキー `kyotsu_v5` で保存されます。  
ブラウザのデータを消去するとリセットされるため、定期的に「データを書き出す」でバックアップしてください。
